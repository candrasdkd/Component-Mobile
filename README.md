## Installaton

---

STEP 1 : Add this folder in local computer

```
git clone https://github.com/candrasdkd/technical_MA.git
```

STEP 2 : Install module & Package

```
yarn install
```



STEP 3 : Type on the terminal/CMD

```
Yarn run android
```
